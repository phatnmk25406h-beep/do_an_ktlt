# M3M English App

## Hướng dẫn cài đặt và chạy ứng dụng

1. Mở Terminal tại thư mục chứa dự án.
2. Tạo môi trường ảo bằng cách ghi dưới chỗ terminal: python -m venv .venv
3. Kích hoạt môi trường:
   - Windows: venv\Scripts\activate
   - Mac/Linux: source .venv/bin/activate
4. Cài đặt thư viện: pip install -r requirements.txt
5. Khởi chạy ứng dụng: vào file main.py và bấm RUN
